# Python Variable Annotations

This project covers the use of type annotations in Python 3.

## Learning Objectives

- Understand type annotations in Python 3
- Specify function signatures and variable types
- Understand duck typing
- Validate code using mypy

## Requirements

- All files interpreted with Python 3.9
- Code follows pycodestyle (version 2.5)
- All modules, classes, and functions are documented
- All files are executable
